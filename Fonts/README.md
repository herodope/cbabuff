# Fonts

Drop the following files here (both are Google Fonts, OFL-licensed,
redistributable) to bring the redesigned UI up to full type-scale fidelity.
Until they exist, `UI/Theme.lua` falls back to stock WoW fonts automatically
— no code change needed once these land.

- `Rajdhani-Regular.ttf` (weight 400)
- `Rajdhani-Medium.ttf` (weight 500)
- `Rajdhani-SemiBold.ttf` (weight 600)
- `Rajdhani-Bold.ttf` (weight 700)
- `Barlow-Light.ttf` (weight 300)
- `Barlow-Regular.ttf` (weight 400)
- `Barlow-Medium.ttf` (weight 500)
- `Barlow-SemiBold.ttf` (weight 600)
- `Barlow-Bold.ttf` (weight 700)
